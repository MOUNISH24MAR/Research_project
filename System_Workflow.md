# System Workflow

## AI-Powered Smart Interview Preparation System

The system follows a structured workflow to provide an intelligent and personalized interview preparation experience.

### Step 1: Resume Upload

The candidate uploads their resume through the platform. The resume serves as the primary source for identifying skills, technologies, educational background, projects, and experience.

⬇️

### Step 2: ATS-Based Resume Analysis

The uploaded resume is analyzed using an Applicant Tracking System (ATS) engine. The system extracts relevant keywords, technical skills, programming languages, tools, certifications, and project details.

**Output:**

* Extracted Skills
* Resume Score
* Skill Gap Analysis

⬇️

### Step 3: Company Selection

The candidate selects a target company such as TCS, Infosys, Zoho, Accenture, Wipro, Amazon, or other organizations.

**Output:**

* Company-specific interview pattern
* Company-specific question database

⬇️

### Step 4: Question Generation

Based on the resume analysis and selected company, the system generates relevant HR and technical interview questions.

**Question Categories:**

* HR Questions
* Technical Questions
* Project-Based Questions
* Aptitude and Problem-Solving Questions

⬇️

### Step 5: Adaptive Mock Interview

The AI interview engine conducts a mock interview. Questions are dynamically adjusted according to the candidate's previous responses and performance level.

**Features:**

* Adaptive Questioning
* Difficulty Adjustment
* Realistic Interview Simulation

⬇️

### Step 6: Speech-to-Text Conversion

Candidate responses are captured through a microphone and converted into text using Speech Recognition technology.

**Output:**

* Transcript of Candidate Responses

⬇️

### Step 7: AI-Based Answer Evaluation

The converted text is evaluated using NLP and BERT-based models.

**Evaluation Metrics:**

* Technical Correctness
* Relevance
* Completeness
* Communication Quality
* Confidence in Response

⬇️

### Step 8: Emotion Detection

The candidate's facial expressions are analyzed using a CNN-based emotion detection model through webcam input.

**Detected Emotions:**

* Confidence
* Nervousness
* Happiness
* Stress
* Neutral State

⬇️

### Step 9: Feedback Generation

The system generates a detailed performance report containing:

* Technical Score
* Communication Score
* Emotion Analysis
* Strengths
* Weaknesses
* Improvement Suggestions

⬇️

### Step 10: Progress Tracking

Interview results are stored in the database to monitor candidate improvement over time.

**Output:**

* Interview History
* Performance Trends
* Skill Improvement Reports
* Personalized Recommendations
